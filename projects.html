MST400 Project 1: Azure VPN and Backup Infrastructure

Designed and implemented a secure Azure-based infrastructure with:

  • Point-to-Site (P2S) VPN tunnel for secure remote access
  • Cross-region file sharing and drive mapping between virtual networks
  • Automated backup schedules: weekly (3-week retention) and monthly (10-month retention)
  • Manual backup and restoration process demonstration
This solution demonstrated practical cloud security, file resiliency, and disaster recovery planning.
